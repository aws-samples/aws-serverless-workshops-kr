Facebook App Id!

No Facebook App Id configured.

In order for your application to establish trust between a federated identity provider we need to provide the appropriate facebook app id to Cognito. That requires 2 steps: 1 add Facebook App Id to environment.ts and 2) add the Facebook App Id to your Cognito Identity Pool.

Go to instructions in README »

Ticket API URL!

No API URL configured.

We need to specify a backend api url in order for this static single page web application to communicate with the ticket service api.

Go to instructions in README »

Invalid Ticket API URL!

API URL needs a trailing "/".

Please check your environment.ts file and make sure the API URL has a trailing forward slash. E.g. https://2xxxxyyyyzz.execute-api.us-east-1.amazonaws.com/prod/
